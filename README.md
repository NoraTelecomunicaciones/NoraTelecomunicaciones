## 🛠️ Servicios

### 📡 Internet (ISP)
Internet **inalámbrico** y **fibra óptica** para hogares y empresas en la Región del Maule.

### 💻 Diseño de software
Desarrollo de **software a medida**: aplicaciones nativas para Linux, herramientas de red, integración
de hardware, soluciones multimedia y automatización.

### 🧭 Consultoría de software
Asesoría técnica: **arquitectura de software, redes e infraestructura**, integración de sistemas y
soluciones multimedia/NDI.
